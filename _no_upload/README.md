# SeederForXF2
This add-on helps in filling development board with dummy data.

## CLI Commands
| Option | Description |
| ------ | ----------- |
| `tck-seeder:seed` | Seeds only a specific seed. |
| `tck-seeder:bulk-seed` | Seeds provided or every seed available. |
